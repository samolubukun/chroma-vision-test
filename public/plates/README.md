# Ishihara Plates Setup

## Instructions

1. Extract the `Plates.zip` file in this directory
2. After extraction, you should have image files named: 24.jpg, 25.jpg, 26.jpg, ..., 54.jpg
3. All image files should be directly in the `public/plates/` folder (not in a subfolder)

The app will automatically load the plates from this location.

## Expected Files

- 24.jpg through 40.jpg (Plates 1-17)
- 45.jpg through 48.jpg (Plates 22-25)
- 51.jpg through 54.jpg (Plates 18-21)

Total: 21 plate images
